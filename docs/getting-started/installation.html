Installation

Before you start using OpenAPI Qraft, you need to install the Qraft binding for the React and the CLI tool to generate the API client. You will also need to install openapi-typescript to generate the types from the OpenAPI Document.

  • npm
  • yarn
  • pnpm
npm install @openapi-qraft/react
npm install -D @openapi-qraft/cli
npm install -D openapi-typescript
yarn add @openapi-qraft/react
yarn add --dev @openapi-qraft/cli
yarn add --dev openapi-typescript
pnpm add @openapi-qraft/react
pnpm add -D @openapi-qraft/cli
pnpm add -D openapi-typescript

Requirements​

OpenAPI Qraft is compatible with React v18+ and works with ReactDOM and React Native. Browser support is the sa as requirements as TanStack Query:

Chrome >= 91
Firefox >= 90
Edge >= 91
Safari >= 15
iOS >= 15
Opera >= 77
Peer Dependency

If you are not already using TanStack Query v5, you need to install it as well:

  • npm
  • yarn
  • pnpm
npm install @tanstack/react-query
yarn add @tanstack/react-query
pnpm add @tanstack/react-query
