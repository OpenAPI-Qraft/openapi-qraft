# Installation

To start using OpenAPI Qraft, you'll need to install two packages:
1. The CLI tool for generating the API client
2. The Qraft Client React package for using the generated API client

```bash npm2yarn
npm install -D @openapi-qraft/cli
npm install @openapi-qraft/react
```

:::info Peer Dependency

If you're not already using **TanStack Query v5**, you'll need to
[install](https://tanstack.com/query/latest/docs/framework/react/installation) it separately:

```bash npm2yarn
npm install @tanstack/react-query
```

:::
