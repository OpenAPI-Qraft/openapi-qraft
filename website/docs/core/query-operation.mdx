# GET, HEAD, OPTIONS

The method allows you to execute a _Queries_ without TanStack's ~~_QueryClient_~~.

<Tabs>
  <TabItem value="with-parameters" label={<span style={{verticalAlign: 'middle'}}>With <code>parameters</code></span>} default>
    ```ts
    const result = qraft.<service>.<operation>(
      {
        parameters,
        baseUrl,
        signal,
        meta,
      },
      requestFn
    );
    ```

    ### Arguments

    1.  - `parameters: { path, query, header } | void`
          - **Required only if OpenAPI specification defines required parameters**
          - If the operation has no required parameters according to OpenAPI, you can omit this argument
        - `baseUrl` - **Required** base URL for the `requestFn`
        - `signal` - An optional `AbortSignal` to cancel the request
        - `meta` - An optional object that will be passed to the `requestFn`
    2.  `requestFn?: RequestFn`
        - **Optional**, a function that will be used to execute the request
  </TabItem>
  <TabItem value="with-query-key" label={<span style={{verticalAlign: 'middle'}}>With <code>queryKey</code></span>}>
    ```ts
    const result = qraft.<service>.<operation>(
      {
        queryKey,
        baseUrl,
        signal,
        meta,
      },
      requestFn
    );
    ```
    ### Arguments

    1.  - `queryKey: QueryKey | void`
          - **Required only if OpenAPI specification defines required parameters**
          - If the operation has no required parameters according to OpenAPI, you can omit this argument
        - `baseUrl` - **Required** base URL for the `requestFn`
        - `signal` - An optional `AbortSignal` to cancel the request
        - `meta` - An optional object that will be passed to the `requestFn`
    2.  `requestFn?: RequestFn`
        - **Optional**, a function that will be used to execute the request.
  </TabItem>
</Tabs>

### Returns

`result: Promise<T>` - The result of the query execution

### Examples

<Tabs>
  <TabItem value="with-parameters" label={<span>With <code>parameters</code></span>} default>
    ```tsx
    import { requestFn } from '@openapi-qraft/react';
    import { createAPIClient } from './api'; // generated by OpenAPI Qraft CLI

    const api = createAPIClient({
      requestFn,
      baseUrl: 'https://petstore3.swagger.io/api/v3',
    });

    /**
     * Executes the request:
     * ###
     * GET /posts?limit=10
     **/
    const posts = await api.posts.getPosts(
      {
        parameters: { query: { limit: 10 } }
      }
    );
    ```
  </TabItem>
  <TabItem value="with-query-key" label={<span>With <code>queryKey</code></span>}>
    ```tsx
    import { requestFn } from '@openapi-qraft/react';
    import { createAPIClient } from './api'; // generated by OpenAPI Qraft CLI

    const api = createAPIClient({
      requestFn,
      baseUrl: 'https://petstore3.swagger.io/api/v3',
    });

    /**
     * Executes the request:
     * ###
     * GET /posts?limit=10
     **/
    const posts = await api.posts.getPosts(
      {
        queryKey: qraft.posts.getPosts.getQueryKey({ query: { limit: 10 } }),
        baseUrl: 'https://custom-api.sandbox.monite.com/v1', // optionally, you can specify the base URL
      }
    );
    ```
  </TabItem>
</Tabs>
